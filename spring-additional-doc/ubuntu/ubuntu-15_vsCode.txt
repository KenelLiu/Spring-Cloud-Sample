开发react web页面，在vs code安装插件工具
ESLint：ESLint 是一个 JavaScript 代码质量工具，可以帮助你保持一致的代码风格，并检测潜在的问题。在 React 项目中使用 ESLint 插件可以确保你的代码质量。

Prettier：Prettier 是一个代码格式化工具，可以自动格式化你的代码，使其保持一致的风格。与 ESLint 结合使用可以实现代码格式和质量的双重保证。

Reactjs code snippets：这个插件提供了大量的 React 代码片段，可以快速生成常见的 React 组件和代码块。

-- Debugger for Chrome：如果你在开发中使用 Chrome 进行调试，这个插件可以帮助你在 VS Code 中调试 React 应用。

React Developer Tools：这不是一个 VS Code 插件，而是一个 Chrome 扩展程序，但它非常有用。它允许你检查 React 组件层次结构，状态和性能分析。

Auto Import：这个插件可以自动导入你在代码中使用的模块，减少手动导入的工作。

-- GitLens：如果你使用 Git 进行版本控制，GitLens 插件可以帮助你更好地理解代码的历史和作者信息。

-- Bracket Pair Colorizer：这个插件可以让括号成对显示，并用不同颜色进行区分，使你更容易理解嵌套的代码块。

Material Icon Theme：这个插件为不同类型的文件添加了漂亮的图标，使你更容易识别它们。

-- Better Comments：通过这个插件，你可以使用不同的注释样式来区分重要性，使注释更加易于理解。

npm Intellisense：这个插件可以为你的项目中的 npm 包提供智能补全功能。
